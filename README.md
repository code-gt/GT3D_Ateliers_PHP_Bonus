# 🚀 Ateliers PHP — Garage404

## 🎯 Objectif

Dans ce cycle d’ateliers, vous allez pratiquer le PHP à travers plusieurs mini-projets.

👉 Chaque atelier permet de travailler :
- les formulaires HTML
- la récupération des données avec `POST`
- les conditions (`if`)
- les variables
- l’affichage dynamique en PHP

---

## 🧠 Organisation

👉 Vous avez le **choix entre 3 ateliers différents**.

Chaque atelier correspond à un **niveau de difficulté** différent :

| Atelier | Thème | Difficulté |
|--------|------|-----------|
| 🐾 TP 1 | Créature fantastique | ⭐ Facile |
| 🎧 TP 2 | Playlist musicale | ⭐⭐ Moyen |
| 🧙 TP 3 | Quiz développeur | ⭐⭐⭐ Difficile |

---

## 🎯 Consigne générale

Pour chaque atelier :

- Créer un formulaire HTML
- Envoyer les données avec la méthode `POST`
- Traiter les données en PHP
- Afficher un résultat dynamique

---

## 🐾 TP 1 — Créature fantastique

### 🧩 Objectif

Créer une fiche de créature personnalisée.

### ✨ À faire

- Créer un formulaire avec :
  - nom
  - type (radio)
  - couleur (select)
  - pouvoir
  - danger
  - habitat

- Afficher une **carte stylée**
- Afficher une image selon le type
- Afficher le niveau de danger avec des ⚠️

---

## 🎧 TP 2 — Playlist musicale

### 🧩 Objectif

Créer une playlist personnalisée.

### ✨ À faire

- Créer un formulaire avec :
  - style musical
  - ambiance
  - durée

- Créer des tableaux de musiques
- Utiliser des conditions pour choisir la playlist
- Utiliser une boucle pour afficher les musiques
- Adapter le nombre de musiques selon la durée

---

## 🧙 TP 3 — Quiz développeur

### 🧩 Objectif

Déterminer ton profil de développeur.

### ✨ À faire

- Créer un quiz avec 5 questions
- Chaque réponse donne des points à un profil :
  - frontend
  - backend
  - game
  - fullstack

- Compter les points
- Trouver le profil dominant
- Afficher :
  - un titre
  - une image
  - une description (3 lignes)

---

## 💡 Conseils

- Commence simple
- Teste souvent ton code
- Utilise `var_dump()` pour déboguer
- N’hésite pas à poser des questions

---

## ⭐ Bonus

Si tu termines rapidement :

- Ajouter des animations CSS
- Ajouter des emojis
- Ajouter des images dynamiques
- Améliorer le design
- Gérer les erreurs (champs vides)

---

## 🚀 Objectif final

👉 Comprendre :
- comment fonctionnent les formulaires en PHP
- comment traiter les données
- comment afficher du contenu dynamique

---

## 🎯 Important
