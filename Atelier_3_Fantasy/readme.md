# 🐾 TP — Créature fantastique

## 🎯 Objectif

Créer une carte dynamique représentant une créature fantastique à partir d’un formulaire.

---

## 📝 Consignes

### 1. Formulaire (index.php)

Créer un formulaire avec :

- Nom de la créature (input text)
- Type (radio)
  - dragon
  - licorne
  - robot
  - alien
- Couleur (input text)
- Pouvoir (input text)
- Danger (input number entre 1 et 5)
- Habitat (select)

---

### 2. Traitement (result.php)

Récupérer les données avec `$_POST`.

---

### 3. Affichage

Afficher une **carte avec flexbox** contenant :

- Image
- Nom
- Type
- Couleur
- Pouvoir
- Danger (avec ⚠️)
- Habitat

---

### 4. Image dynamique

Afficher une image différente selon le type :

- dragon.jpg
- licorne.jpg
- robot.jpg
- alien.jpg

---

## 🧠 Notions travaillées

- Variables
- Conditions (if / elseif)
- Formulaires
- POST
- Affichage dynamique
- HTML + CSS
- Flexbox
